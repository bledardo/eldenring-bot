Add the following 4 functions to storage.js after the existing ER Betting section (after `getAllActiveEldenRingBets`), under a new comment block `// ELDEN RING STATS`. Export all 4 in module.exports under a new `// Elden Ring Stats` comment.

1. **getAllEldenRingPlayerIds()**
   - `const data = readData();`
   - Return `Object.keys(data.eldenRing?.players || {})`

2. **getEldenRingPlayerStats(discordId)**
   - `const data = readData();`
   - Get `data.eldenRing?.players?.[discordId]` — if null, return null
   - Iterate `Object.entries(player.bosses || {})`:
     - For each boss, count kills (outcome === 'kill'), deaths (outcome === 'death')
     - Sum `duration_seconds` for fight time
     - Track if boss was defeated (kills > 0)
     - Build per-boss entry: `{ bossName, attempts: fights.length, kills, deaths, defeated, fightTime, lastFight: fights[fights.length - 1] || null }`
   - Iterate `player.sessions || []`:
     - For sessions with both `start` and `end`, sum `(new Date(end) - new Date(start)) / 1000` for total session time
   - Return object:
     ```javascript
     {
       totalKills,      // sum of all boss kills
       totalDeaths,     // sum of all boss deaths
       bossesDefeated,  // count of bosses with at least one kill
       bossesEncountered, // count of distinct bosses
       totalFightTime,  // sum of all duration_seconds across all fights
       totalSessionTime, // sum of all session durations
       bosses: bossEntries, // array of per-boss objects
     }
     ```

3. **getEldenRingLeaderboard()**
   - `const data = readData();`
   - Get `data.eldenRing?.players || {}`
   - For each player (discordId, playerData): iterate all bosses and sum kills, deaths, and totalTime (duration_seconds)
   - Build entry: `{ discordId, kills, deaths, totalTime, bossesDefeated }`
   - Return:
     ```javascript
     {
       byKills: [...entries].sort((a, b) => b.kills - a.kills),
       byDeaths: [...entries].sort((a, b) => b.deaths - a.deaths),
       byTime: [...entries].sort((a, b) => b.totalTime - a.totalTime),
     }
     ```

4. **getEldenRingBossDifficulty()**
   - `const data = readData();`
   - Get `data.eldenRing?.players || {}`
   - For each player, for each boss, count deaths (outcome === 'death')
   - Accumulate `bossDeaths[bossName] += deaths`
   - Return array sorted descending by deaths:
     ```javascript
     Object.entries(bossDeaths)
       .map(([bossName, deaths]) => ({ bossName, deaths }))
       .sort((a, b) => b.deaths - a.deaths)
     ```

Add all 4 functions to `module.exports` at the bottom of the file, after the existing `// Elden Ring Betting` exports:
```javascript
  // Elden Ring Stats
  getAllEldenRingPlayerIds,
  getEldenRingPlayerStats,
  getEldenRingLeaderboard,
  getEldenRingBossDifficulty,
```
