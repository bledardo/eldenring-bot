  <files>~/delu-bot/src/storage.js, ~/delu-bot/tests/eldenRingStorage.test.js</files>
  <action>
Add the following functions to storage.js, placed after the existing ER functions (after cleanEldenRingSeenEvents) and exported in module.exports:

1. **createEldenRingBet(fightId, discordUserId, bossName, messageId, channelId)**
   - Stores under `data.eldenRing.activeEldenRingBets[fightId]`
   - Structure: `{ discordUserId, bossName, messageId, channelId, bets: {}, hasGoldenOffer: Math.random() < 0.2, firstBettorId: null, closedAt: null, createdAt: new Date().toISOString() }`
   - Ensure `data.eldenRing.activeEldenRingBets` object exists (create if missing)
   - Return `{ hasGoldenOffer }`

2. **getEldenRingBet(fightId)**
   - Returns `data.eldenRing.activeEldenRingBets[fightId]` or null

3. **getActiveEldenRingFightId(discordUserId, bossName)**
   - Scans `activeEldenRingBets` for entry where `discordUserId` matches AND `closedAt` is null AND `bossName` matches
   - Returns fightId string or null

4. **getEldenRingBossDefeatRate(discordId, bossName)**
   - Uses existing `getEldenRingBossFights(discordId, bossName)`
   - If no fights, return 0.75 (default: 75% defeat rate for unknown boss)
   - Otherwise: `deaths / totalFights` where deaths = fights with `outcome === 'death'`

5. **calculateEldenRingOdds(discordId, bossName)**
   - Gets defeat rate via `getEldenRingBossDefeatRate()`
   - Passes `1 - defeatRate` as winrate to existing `calculateBaseOdds()`
   - Returns `{ victoireOdds: winOdds, defaiteOdds: loseOdds, defeatRate }`
   - victoireOdds = odds for "player kills boss" bet, defaiteOdds = odds for "player dies" bet

6. **placeEldenRingBet(fightId, odUserId, prediction, amount)**
   - prediction is 'victoire' or 'defaite'
   - Follow existing placeBet() pattern: check bet exists, check not closed, check wallet balance, deduct amount
   - Create wallet with getStartingBalance() if missing (same pattern as existing placeBet)
   - If user already bet on different prediction, reject: "Vous avez deja parie sur l'autre resultat"
   - If user already bet same prediction and bet_modification disabled, reject
   - Calculate locked odds via calculateEldenRingOdds() using the bet's discordUserId + bossName
   - Store: `bet.bets[odUserId] = { prediction, amount, lockedOdds }`
   - Set `bet.firstBettorId = odUserId` if first_bettor_bonus feature enabled and firstBettorId is null
   - Return `{ success: true, newBalance, lockedOdds }` or `{ success: false, message }`

7. **closeEldenRingBet(fightId, result)**
   - result is 'victoire' or 'defaite'
   - Follow existing closeBet() pattern for wallet math
   - For each bettor: if prediction matches result -> winner, else loser
   - Winners: credit `amount * lockedOdds` to wallet (round down with Math.floor)
   - Golden Offer: if hasGoldenOffer and winner's amount >= 100, double the profit portion
   - First bettor bonus: if firstBettorId matches winner, add 10% bonus on profit
   - Losers: update totalLost
   - Update leaderboard wins/losses/streak
   - Set `bet.closedAt = new Date().toISOString()`
   - Delete from activeEldenRingBets after processing
   - Return `{ winners: [{odUserId, amount, profit, lockedOdds}], losers: [{odUserId, amount}] }`

8. **cancelEldenRingBet(fightId)**
   - Refund all bettors: credit back their amount to wallet
   - Delete from activeEldenRingBets
   - Return count of refunded bets

Also add to `ensureDataFile()` migration: if `!data.eldenRing.activeEldenRingBets`, create it as `{}`.

Export all 8 new functions in module.exports.
