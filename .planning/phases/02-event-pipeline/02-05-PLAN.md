  <name>Task 1: Wire eldenRingTracker into index.js and connect apiServer to EventEmitter</name>
  <files>~/delu-bot/src/index.js, ~/delu-bot/src/apiServer.js</files>
  <action>
**1. Update index.js:**

Add import at top (after tftTracker import):
```javascript
const eldenRingTracker = require('./eldenRingTracker');
```

In the `main()` function, after `tftTracker.start(client);`, add:
```javascript
// Start Elden Ring tracker (handles feature flag check internally)
eldenRingTracker.start(client);
```

In both SIGINT and SIGTERM handlers, after `tftTracker.stop();`, add:
```javascript
eldenRingTracker.stop();
```

**2. Update apiServer.js event handlers to emit events:**

In each event handler function (handleBossEncounter, handlePlayerDeath, handleBossKill, handleSessionStart, handleSessionEnd), add the emitEvent call AFTER the storage call:

```javascript
// At top of apiServer.js, add lazy import:
let _tracker = null;
function getTracker() {
  if (!_tracker) _tracker = require('./eldenRingTracker');
  return _tracker;
}

// In each handler, after storage call:
// handleBossEncounter:
getTracker().emitEvent('boss_encounter', { discordUserId, bossName: data.boss_canonical_name || data.boss, attemptNumber: fight.attempt_number, sessionId: data.session_id, timestamp: data.timestamp });

// handlePlayerDeath:
getTracker().emitEvent('player_death', { discordUserId, bossName: data.boss_canonical_name || data.boss, attemptNumber: fight.attempt_number, sessionId: data.session_id, timestamp: data.timestamp });

// handleBossKill:
getTracker().emitEvent('boss_kill', { discordUserId, bossName: data.boss_canonical_name || data.boss, attemptNumber: fight.attempt_number, sessionId: data.session_id, timestamp: data.timestamp });

// handleSessionStart:
getTracker().emitEvent('session_start', { discordUserId, sessionId: data.session_id, timestamp: data.timestamp });

// handleSessionEnd:
getTracker().emitEvent('session_end', { discordUserId, sessionId: data.session_id, timestamp: data.timestamp, summary: data.summary });
```

Use lazy import (`getTracker()`) to avoid circular dependency since eldenRingTracker requires apiServer and vice versa.

**CRITICAL:** Do not modify the existing tracker.start(client) or tftTracker.start(client) calls. Add eldenRingTracker AFTER them.
