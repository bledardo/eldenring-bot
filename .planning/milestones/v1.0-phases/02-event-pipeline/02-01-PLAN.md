<task type="auto">
  <name>Task 2: Extend storage.js with Elden Ring data namespace and CRUD functions</name>
  <files>~/delu-bot/src/storage.js</files>
  <action>
1. In `ensureDataFile()`, add migration block after the `spontaneousBets` migration (around line 245):
```javascript
if (!data.eldenRing) {
  data.eldenRing = {
    apiKeys: {},      // { discordId: { key, created_at } }
    players: {},      // { discordId: { bosses: {}, sessions: [] } }
    seenEventIds: {}, // { uuid: timestamp_ms } for dedup
  };
  needsWrite = true;

```

2. Add a new section `// ============================================\n// ELDEN RING\n// ============================================` before the FEATURE FLAGS section (around line 2667). Add these functions:

**API Key management:**
- `getEldenRingApiKey(discordId)` — returns `data.eldenRing.apiKeys[discordId]` or null
- `setEldenRingApiKey(discordId, keyObj)` — sets `data.eldenRing.apiKeys[discordId] = keyObj`, writes data
- `getEldenRingPlayerByApiKey(apiKey)` — iterates `data.eldenRing.apiKeys`, returns discordId whose `.key === apiKey`, or null
- `deleteEldenRingApiKey(discordId)` — deletes `data.eldenRing.apiKeys[discordId]`, writes data

**Fight data:**
- `getEldenRingPlayer(discordId)` — returns `data.eldenRing.players[discordId]` or null
- `ensureEldenRingPlayer(discordId)` — creates `{ bosses: {}, sessions: [] }` if not exists, returns player
- `addEldenRingFight(discordId, bossName, fight)` — ensures player, ensures `bosses[bossName] = { fights: [] }` if not exists, pushes fight, writes data. Returns the fight with `attempt_number` set to the new length of `fights[]`.
- `getEldenRingBossFights(discordId, bossName)` — returns fights array or empty array

**Session management:**
- `startEldenRingSession(discordId, sessionId, timestamp)` — ensures player, pushes `{ id: sessionId, start: timestamp, end: null, summary: null }` to sessions, writes data
- `endEldenRingSession(discordId, sessionId, timestamp, summary)` — finds session by id, sets end and summary, writes data

**Deduplication:**
- `isEldenRingEventSeen(eventId)` — returns `!!data.eldenRing.seenEventIds[eventId]`
- `markEldenRingEventSeen(eventId)` — sets `data.eldenRing.seenEventIds[eventId] = Date.now()`, writes data
- `cleanEldenRingSeenEvents(maxAgeMs = 24 * 60 * 60 * 1000)` — removes entries older than maxAgeMs, writes data if any removed

3. Add all new functions to `module.exports` at the bottom of the file.

Follow the existing code style: use `readData()`/`writeData()` pattern, no async, French comments where natural but function names in English (matching existing pattern).
