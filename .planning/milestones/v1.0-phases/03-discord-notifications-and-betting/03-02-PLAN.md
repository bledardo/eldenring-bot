Create `~/delu-bot/src/eldenRingNotifier.js` with a `register(client)` function that subscribes to eldenRingEvents.

**Module structure:**
```javascript
const { EmbedBuilder, ActionRowBuilder, ButtonBuilder, ButtonStyle } = require('discord.js');
const { eldenRingEvents } = require('./eldenRingTracker');
const storage = require('./storage');
const { getEldenRingBossImage } = require('./eldenRingAssets');
const crypto = require('crypto');

function register(client) {
  eldenRingEvents.on('boss_encounter', (data) => handleEncounter(client, data));
  eldenRingEvents.on('player_death',   (data) => handleDeath(client, data));
  eldenRingEvents.on('boss_kill',      (data) => handleKill(client, data));
  // session_end handled in Plan 04
}

module.exports = { register };
```

**handleEncounter(client, { discordUserId, bossName, attemptNumber, timestamp }):**

1. Check betting_elden_ring feature flag — if disabled, post embed without buttons
2. Generate short fightId: `er_${crypto.randomBytes(4).toString('hex')}` (8 hex chars, stays under Discord 100-char customId limit per Pitfall 5 in research)
3. Cancel any existing open bet for same discordUserId + bossName (Pitfall 2: orphaned bets)
4. Build gold/amber embed (color 0xFFA500):
   - Title: `"⚔️ Un Grand Ennemi est apparu..."` (per user decision: dramatic narrator energy)
   - Description: Include boss name in bold, dramatic boss introduction text. Use a `getEpicIntro(bossName, attemptNumber)` helper that varies text by attempt number (first time: "steps into the arena", repeat: "returns to face their nemesis")
   - Fields:
     - `Tarnished`: `<@${discordUserId}>` (inline)
     - `Tentative`: `#${attemptNumber}` (inline)
     - `Paris`: `0 Victoire / 0 Defaite` (inline)
   - Image: `embed.setImage(getEldenRingBossImage(bossName))` — only if not null (per user decision: skip entirely if no artwork)
   - Timestamp: current time
   - Footer: odds info (e.g., "Cotes: Victoire x2.50 / Defaite x1.40")
5. Build 2 ActionRows with 6 buttons total (per user decision: predefined bet tiers 50/100/500):
   - Row 1 (Victoire): 3 green buttons
     - `er_bet_v_50_${fightId}` label "50 PC Victoire" style Success
     - `er_bet_v_100_${fightId}` label "100 PC Victoire" style Success
     - `er_bet_v_500_${fightId}` label "500 PC Victoire" style Success
   - Row 2 (Defaite): 3 red buttons
     - `er_bet_d_50_${fightId}` label "50 PC Defaite" style Danger
     - `er_bet_d_100_${fightId}` label "100 PC Defaite" style Danger
     - `er_bet_d_500_${fightId}` label "500 PC Defaite" style Danger
   - NOTE: Use short prefixes `v`/`d` instead of full `victoire`/`defaite` to keep customId under 100 chars
6. Send embed to notification channel (storage.getNotificationChannelId())
7. Call storage.createEldenRingBet(fightId, discordUserId, bossName, message.id, channelId)
8. If hasGoldenOffer, edit embed to add gold star in footer: "🌟 GOLDEN OFFER — Gains doubles!"
9. Wrap everything in try/catch with `console.warn('[ER Notifier]', e.message)` — never crash the event handler (per research anti-pattern)

**handleDeath(client, { discordUserId, bossName, attemptNumber, timestamp }):**

1. Get notification channel
2. Build red embed (color 0xFF0000):
   - Title: death message that varies with attempt count via `getDeathTaunt(attemptNumber)`:
     - 1-3: Light taunts ("Le Ternis a chuté..." / "Encore une chute...")
     - 4-7: Medium ("Ce boss ne rigole pas..." / "La patience est mise à rude épreuve...")
     - 8-14: Heavy ("A ce stade, c'est le boss qui te farm..." / "Les murs tremblent sous les ragequits...")
     - 15+: Savage ("Tu es devenu un distributeur d'âmes..." / "Le boss s'ennuie tellement qu'il bâille entre tes tentatives...")
     - Create 3-4 messages per tier, pick randomly for variety
   - Description: `<@${discordUserId}> est tombé face à **${bossName}**`
   - Fields:
     - `Tentative`: `#${attemptNumber}` (inline)
   - Boss image if available
   - Timestamp
3. Send to channel
4. Wrap in try/catch

**handleKill(client, { discordUserId, bossName, attemptNumber, timestamp, durationSeconds }):**

1. Get notification channel
2. Build green embed (color 0x00CC44):
   - Title: triumphant message (per user decision: match dramatic tone of encounter)
     - `"🏆 BOSS VAINCU !"` for standard kills
     - `"🏆 FIRST TRY ! BOSS VAINCU !"` if attemptNumber === 1
   - Description: `<@${discordUserId}> a terrasse **${bossName}** !`
   - Fields:
     - `Tentatives`: `${attemptNumber}` (inline)
     - `Duree du combat`: formatted duration from durationSeconds (e.g., "2m 34s") (inline)
   - Boss image if available
   - Timestamp
3. Send to channel
4. Wrap in try/catch

NOTE: Bet resolution logic (closing bets on death/kill, updating encounter embed) is handled in Plan 04. This plan ONLY posts the event embeds and creates the initial bet state. The death/kill handlers here do NOT close bets — that will be added as a separate concern in Plan 04.

**Helper functions (internal to module):**
- `getEpicIntro(bossName, attemptNumber)` — dramatic flavor text
- `getDeathTaunt(attemptNumber)` — escalating death messages
- `formatDuration(seconds)` — "Xm Ys" or "Xh Ym" format
- `getNotifChannel(client)` — get channel from cache, return null if not found
